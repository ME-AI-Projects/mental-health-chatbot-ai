# 🧠 AI Mental Health Assistant

An AI-powered chatbot designed to provide **empathetic and supportive responses** using a hybrid retrieval system and safety-aware processing.

---

## 🚀 Features

- 💬 Conversational mental health support
- 🔍 Hybrid Retrieval (TF-IDF + E5 Embeddings)
- ⚠️ Risk Detection (ML + rule-based)
- 🛡️ Safety Filtering
- 🎯 Response Ranking & Rewriting
- 🧠 Emotion Detection

---

## 🏗️ System Architecture

User Input → Preprocessing →  
Hybrid Retrieval (TF-IDF + E5) →  
Ranking → Safety Filter →  
Response Rewriting → Final Output  

---

## 🔍 How It Works

1. **User Query**
2. Context-aware query building
3. Retrieve top responses using:
   - TF-IDF (keyword matching)
   - E5 embeddings (semantic similarity)
4. Rank responses
5. Apply safety filters
6. Rewrite response to be empathetic
